Git

  • git status - Показывает состояние рабочего каталога и проиндексированного снимка состояния. Эту команду можно выполнять в сочетании с git add и git commit, чтобы узнать, что именно будет включено в следующий снимок.
  • git add - Переносит изменения из рабочего каталога в раздел проиндексированных файлов..
  • git commit -m - Коммит — это операция по добавлению всех проиндексированных файлов в репозиторий командой git commit –m “changes message”.
  • git push - это одна из консольных команд Git. Она позволяет передать изменения из локального репозитория (набора файлов из папки . git) в удаленный.
  • git log - перечисляет коммиты, сделанные в репозитории в обратном к хронологическому порядке — последние коммиты находятся вверху.
  • git pull - для извлечения и загрузки содержимого из удаленного репозитория и немедленного обновления локального репозитория этим содержимым.
